ADMIN CREDENTIALS SETUP
=======================
Packages: BCrypt.Net-Next 4.2.0, Otp.NET 1.4.1

The app reads two keys from appsettings.json:
  "AdminPasskeyHash"  — BCrypt hash of the passkey
  "AdminTotpSecret"   — base32 TOTP shared secret

Both must be correct to log in.


STEP 1 — Generate a new passkey hash
--------------------------------------
Run this temp project to hash your chosen passkey (replace "your-passkey"):

  $tmpDir = "$env:TEMP\cpn-credentials"
  New-Item -ItemType Directory -Force -Path $tmpDir | Out-Null

  [System.IO.File]::WriteAllText("$tmpDir\gen.csproj", @"
  <Project Sdk="Microsoft.NET.Sdk">
    <PropertyGroup>
      <OutputType>Exe</OutputType>
      <TargetFramework>net8.0</TargetFramework>
    </PropertyGroup>
    <ItemGroup>
      <PackageReference Include="BCrypt.Net-Next" Version="4.2.0" />
      <PackageReference Include="Otp.NET" Version="1.4.1" />
    </ItemGroup>
  </Project>
  "@)

  [System.IO.File]::WriteAllText("$tmpDir\Program.cs",
    'var hash = BCrypt.Net.BCrypt.HashPassword("your-passkey", 11); System.Console.WriteLine("HASH:" + hash);'
  )

  dotnet run --project "$tmpDir\gen.csproj"
  Remove-Item -Recurse -Force $tmpDir

Copy the printed hash into appsettings.json -> "AdminPasskeyHash".


STEP 2 — Generate a new TOTP secret
--------------------------------------
Same temp project approach, swap Program.cs content:

  [System.IO.File]::WriteAllText("$tmpDir\Program.cs",
    'var key = OtpNet.KeyGeneration.GenerateRandomKey(20); System.Console.WriteLine("SECRET:" + OtpNet.Base32Encoding.ToString(key));'
  )

  dotnet run --project "$tmpDir\gen.csproj"
  Remove-Item -Recurse -Force $tmpDir

Copy the printed secret into appsettings.json -> "AdminTotpSecret".


STEP 3 — Add secret to authenticator app (one-time)
------------------------------------------------------
Open Google Authenticator / Authy / Microsoft Authenticator:
  -> "Enter a setup key manually"
  -> Account name : Carnivorous Plant Nursery (or anything)
  -> Key          : <the base32 secret from Step 2>
  -> Type         : Time-based

After this the app shows a 6-digit code that rotates every 30 seconds.
The TOTP secret never needs to change unless you re-enroll a new device.
